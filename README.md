# Password-Generator
Create an application that generates a random password based on user-selected criteria. This app will run in the browser and feature dynamically updated HTML and CSS powered by your JavaScript code. It will also feature a clean and polished user interface and be responsive, ensuring that it adapts to multiple screen sizes.

# installation
write Javascript code that intereact with user input to generate a random password.

# Usage
 * press the "Generate Password" button to activate the prompt for user input;

 * ask for different selection criteria from the user:
  
  * if user criteria were met, then javascript will generate a random password pased on user input

  * if user criteria were not met, alert user to try again.
